# trrywebapp

TRRY Apparel public customer web app built with Next.js App Router for Vercel.

## Structure

- `app/layout.jsx` - root metadata and layout
- `app/page.jsx` - public homepage route
- `app/globals.css` - TRRY neo-brutalist design system
- `components/` - reusable UI and app-flow components
- `data/mockData.js` - local MVP product and service data

## Run locally

```bash
npm.cmd run dev
```

## Build

```bash
npm.cmd run build
```
